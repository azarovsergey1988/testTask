
CodeceptJS tests
------------------


### Тербования
 * `Node` version >= 8.9
 * `Java` verion >= 8


Как запустить тесты
--------------------------------------------------------------------------------------------------------

1. Склонировать проект git clone https://github.com/azarovsergey1988/testTask.git
2. Установить npm i codeceptjs -g
3. Перейти в дерикторию проекта и запустить 'npm i '
4. Запустить тесты 'npm run runTests'
5. После запуска тестов посомтреть allure report: 'allure serve output'
